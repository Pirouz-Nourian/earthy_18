This folder contains 1 Rhino example file and 1 grasshopper script.

- The Rhino file has a 6 points.

- The GH file does dynamic relaxation.

The GH file has 2 parts; 1 for relaxing of a curve between 2 points, and 1 for relaxing of a brep made between points.

Plugins required: Pufferfish, Lunchbox.

The GH-definitions have instructions included.
